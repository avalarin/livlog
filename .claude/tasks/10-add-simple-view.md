Нужно добавить на главный экран возможность переключаться между grid и list. Сейчас реализован grid, нужно добавить еще list.

Функциональные требования:
- Пользователь может переключаться между двумя представление;
- Последнее выбранное представление сохраняется между перезапусками приложения;
- В представлении list описание записи видно в 2 строки:
    - слева иконка коллекции, она занимает обе строки
    - в первой строке видно название записи выделенным шрифтом
    - во второй строке видно детали: год, жанр и прочее
    
Визуальные требования:
- Кнопка переключения расположеня вверху, рядом с кнопкой настроек;
- По нажатию происходит переключение между 2 режимами;
- Иконка должна показывать текущее представление: grid или list.
