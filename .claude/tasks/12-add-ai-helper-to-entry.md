Добавить AI помощника на экран создания/редактирования записи (AddEntryView), который по типу коллекции и названию находит несколько вариантов (до 5), предлагает пользователю выбрать нужный вариант и автоматически заполняет все поля записи включая метаданные и фотографии.

## Функциональные требования

### Основной workflow:
1. Пользователь вводит название записи и выбирает коллекцию
2. Нажимает кнопку AI помощника
3. Система показывает sheet с индикатором загрузки и текстом "Ищу варианты..."
4. API находит до 5 наиболее релевантных вариантов по названию и типу коллекции
5. Система показывает bottom sheet со списком найденных вариантов
6. Пользователь выбирает нужный вариант
7. Система автоматически заполняет все поля:
   - Описание (Description)
   - Дополнительные поля (Year, Genre, Author, Platform в зависимости от типа)
   - Фотографии (загружает и добавляет до 3 изображений)
8. Sheet закрывается, пользователь видит заполненную форму

### Обработка ошибок:
- **Если не найдено ни одного варианта**: Показать alert с сообщением "Ничего не найдено, попробуйте другое название"
- **Если ошибка API**: Показать alert с описанием ошибки
- **Если ошибка загрузки фото**: Заполнить остальные поля, но показать уведомление об ошибке загрузки фото

### Условия активации:
- Кнопка AI помощника должна быть активна только если:
  - Выбрана коллекция
  - Введено название (минимум 2 символа)

## Визуальные требования

### Кнопка AI помощника:
- **Расположение**: Рядом с текстовым полем названия (справа от поля)
- **Иконка**: Магический sparkles символ (SF Symbol: `sparkles` или `wand.and.stars`)
- **Стиль**: Кнопка с иконкой, акцентный цвет
- **Состояния**:
  - Disabled (серая) если коллекция не выбрана или название пустое
  - Active (синяя) если условия выполнены
  - Loading (с ProgressView) во время поиска

### Sheet с прогрессом загрузки:
- Модальный sheet с затемненным фоном
- ProgressView в центре
- Текст "Ищу варианты..." под индикатором
- Нельзя закрыть пока идет загрузка

### Bottom Sheet со списком вариантов:
- Модальный sheet снизу с скругленными углами
- Заголовок: "Выберите вариант"
- Список карточек вариантов (до 5), каждая показывает:
  - **Превью картинки**: слева, квадратное 60x60 с rounded corners
  - **Название и год**: жирный шрифт для названия, обычный для года в скобках
  - **Жанр/Автор**: вторая строка с дополнительной информацией (в зависимости от типа коллекции)
  - **Краткое описание**: 2-3 строки описания с многоточием если длинное
- При нажатии на карточку: анимация выбора, затем автозаполнение и закрытие sheet
- Кнопка "Отмена" внизу sheet

### Визуальный стиль карточек вариантов:
- Белый фон с легкой тенью
- Скругленные углы (12pt)
- Padding 12pt внутри карточки
- Spacing 8pt между карточками в списке
- Tap highlight эффект при нажатии

## Дополнительный контекст

Эта фича похожа на существующий SmartAddEntryView, но интегрирована в ручной процесс создания записи. Основные отличия:
- SmartAddEntryView - полностью AI-driven flow с автоматическим созданием
- Новая фича - помощник в ручном процессе, пользователь выбирает что применить

Можно переиспользовать логику из SmartAddEntryView для:
- API calls к OpenRouter
- Загрузки и обработки изображений
- Парсинга JSON ответов
