# Пользовательские сценарии

## Терминология

- Записи — каждая запись, это информация о фильме, книге или др. У записи есть заголовок, набор фото, дата добавления, статус (done, backlog), оценка и коллекция к которой она относится (книга, фильм). 
- Коллекции - группы записей, напр. фильмы или книги.

## Основные

Пользователь работает с записями:
- может просмотреть список записей во всех коллекциях или отфильтровать по одной коллекции;
- может добавлять запись вручную, открыть экран добавления и заполнить все поля, добавить фото;
- может удалять и редактировать записи, в т.ч. переносить записи между коллекциями;
- при создании записи может нажать кнопку "помощь AI" и LLM заполнит недостающие данные автоматически;
- может создавать записи полностью автоматически через bulk режим через ввод текста или фото: AI-агент должен будет распознать текст/фото и предложить создание N записей.

Пользователь работает с коллекциями:
- может создавать коллекции, при создании нужно выбрать иконку и название коллекции;
- может удалять коллекции, при удалении удаляются все записи в коллекции;
- может делиться коллекциями: сгенерировать ссылку-приглашение, по которой другой пользователь сможет получить доступ к этой-же коллекции на редактирование или только на просмотр.
